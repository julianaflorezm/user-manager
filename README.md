# user-manager-api
<p>API buit on hexagonal architecture and DDD as desing pattern</p>

## Requirements
1. Nodejs v18.15
2. Docker v20.10
3. PgAdmin 4 v6
4. Postman

<h4>Database to up</h4>

<pre class="language-markup">
	docker-compose up
</pre>

<h4>Database to down</h4>

<pre class="language-markup">
	docker-compose down
</pre>

<h4>Install dependencies</h4>

<pre class="language-markup">
	npm install
</pre>

<h4>Start project</h4>

<pre class="language-markup">
	npm run start:dev
</pre>

<h4>Run unit test</h4>

<pre class="language-markup">
	npm run test:unit
</pre>

<h4>Run unit test</h4>

<pre class="language-markup">
	npm run test:e2e
</pre>

<h4>Find api documentation</h4>
<p>http://localhost:3000/user-manager/doc</p>
